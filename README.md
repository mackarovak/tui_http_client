# 🗂 HTUI

`HTUI` - это терминальный HTTP-клиент на Go. Проект задуман как легковесная, keyboard-first альтернатива Postman для базовой работы с API: создать запрос, сохранить его локально, отправить и посмотреть ответ, не выходя из терминала.

Сейчас репозиторий содержит: каркас приложения, адаптивный layout, sidebar со списком запросов, локальное JSON-хранилище, слой сборки и выполнения HTTP-запросов, валидацию и экран помощи.

## Что это за проект

Это TUI-приложение на базе:

- `bubbletea` - цикл приложения и управление состоянием
- `bubbles` - UI-компоненты
- `lipgloss` - стили и компоновка
- стандартный `net/http` - выполнение HTTP-запросов

Проект рассчитан на быстрые повседневные сценарии работы с API:

- тестовые `GET`, `POST`, `PUT`, `PATCH`, `DELETE` запросы
- локальное сохранение переиспользуемых запросов
- небольшой список сохраненных запросов под рукой
- поиск, дублирование и удаление запросов
- подготовка body, headers, params и Bearer token

## Что умеет

### Sidebar и список запросов

- показывает сохраненные запросы
- отображает имя запроса, HTTP-метод и сокращенный URL
- фильтрует список по имени или URL
- открывает выбранный запрос
- создает новые запросы
- дублирует и удаляет запросы

### HTTP

- автоматическое добавление `https://`, если схема не указана
- сборку query-параметров
- headers
- режимы body:
  - `none`
  - `raw`
  - `json`
  - `form`
- Bearer authentication
- timeout
- сбор метаданных ответа

### Валидация

В проекте уже есть проверка:

- пустого URL
- невалидного URL
- невалидного JSON
- предупреждение при пустом Bearer token


## Как запустить

- Go `1.25.3` или совместимое окружение

Из корня проекта:

```bash
go mod tidy
go run ./cmd/app
```

Сборка бинарника:

```bash
go mod tidy
go build -o bin/htui ./cmd/app
```

Запуск тестов:

```bash
go mod tidy
go test ./...
```
## Как пользоваться

### Базовый сценарий

1. Запустите приложение командой `go run ./cmd/app`.
2. При первом запуске приложение создаст demo-набор запросов.
3. Используйте левую панель для навигации по сохраненным запросам.
4. Откройте справку клавишей `?`.
5. Создавайте, ищите, дублируйте и удаляйте запросы через sidebar.
